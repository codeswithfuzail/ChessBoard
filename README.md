# Chessboard ♟️

A clean, responsive, and visually accurate Chessboard interface built from scratch using HTML and CSS. This project demonstrates strong foundational skills in layout structuring, CSS Grid/Flexbox alignment, and precise styling.

## 🚀 Features
* **Perfect Grid Layout:** Utilizes efficient CSS layout techniques to build a seamless 8x8 grid.
* **Fully Responsive:** Adapts beautifully across various screen sizes.
* **Modern Design:** Uses a clean color palette for a professional and sleek chess interface.
* **Clean & Semantic Code:** Structured with well-organized HTML and highly maintainable CSS.

## 🛠️ Tech Stack
* **HTML5:** For core page structure and semantic elements.
* **CSS3:** For customized grids, styling, and responsiveness.
